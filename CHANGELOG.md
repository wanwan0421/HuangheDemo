## [1.0.0] - 2026-03-09

### ✨ 新增功能 | Added

#### 核心功能
- 🤖 **智能对话系统**
  - 实现基于 SSE 的流式对话
  - 支持 Markdown 格式消息渲染
  - 集成多智能体工具调用链
  - 实现会话历史管理和切换

- 🗺️ **地图可视化系统**
  - 集成 Mapbox GL JS 地图引擎
  - 支持矢量数据展示（点、线、面）
  - 支持栅格数据展示（GeoTIFF）
  - 实现多图层叠加显示
  - 自动样式化和颜色分配
  - 交互式图例和数据查询

- 📤 **数据处理系统**
  - 支持多种地理数据格式上传
  - 实现文件格式自动转换
  - 完成数据扫描和元数据提取
  - 支持 Shapefile、GeoJSON、GeoTIFF 等格式

- 🎯 **模型推荐系统**
  - 基于需求的智能模型推荐
  - 可视化展示模型工作流
  - 任务需求结构化展示
  - 模型契约验证

#### 页面开发
- 🏠 **首页 (Home)**
  - 轮播图展示系统
  - 项目统计数据展示
  - 响应式布局设计

- 💬 **智能决策 (Decision)**
  - AI 对话界面
  - 文件上传管理
  - 地图可视化集成
  - 扫描结果展示

- 📚 **资源管理 (Resources)**
  - 四级分类菜单
  - 关键字搜索功能
  - 资源详情展示
  - 分页加载

- 📊 **指标体系 (Index)**
  - 指标分级展示
  - 指标关联查询

#### 组件开发
- 📦 **通用组件**
  - MapboxViewer: 地图可视化组件
  - ChatInput: 聊天输入框
  - ToolTimeline: 工具执行时间线
  - TaskSpecCard: 任务规格卡片
  - ModelContract: 模型契约卡片
  - AnimatedContent: 动画内容包装器

#### 技术架构
- ⚡ 采用 React 19 + TypeScript
- 🎨 集成 Tailwind CSS 4.x
- 🔧 使用 Vite 5.x 构建工具
- 🎭 集成 Framer Motion 动画库
- 🗺️ 集成 Mapbox GL JS 3.19

## 版本说明

### 版本号格式
版本号遵循语义化版本规范：`主版本号.次版本号.修订号`

- **主版本号**：重大架构变更或不兼容的 API 修改

- **修订号**：向下兼容的问题修复
